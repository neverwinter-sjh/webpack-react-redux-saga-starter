# Webpack React Redux Saga Starter

React, Redux, React Router, Redux-saga로 설정된 starter boilderplate입니다.

* CRA 사용하지 않고 Webpack 및 Node 설정으로 제작
* react-dev-utils를 이용하여 CRA의 메세지와 최대한 유사하게 log가 나오도록 수정

## install
```
npm install
```

## development
```
npm start
or
npm run dev
```

## build
```
npm run build
```
